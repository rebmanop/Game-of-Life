# Game-of-Life
Simple Game of Life clone

#### Implemented features:
- Adjustable simulation speed
- Explanation window
- Generation and population counters

 1) Download or clone this repository.
 2) Extract to some location.
 3) Pip all dependencies.
 4) Run main.py
